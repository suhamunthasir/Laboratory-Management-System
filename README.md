# Laboratory Management System

## Overview
A full-stack desktop application designed to automate healthcare diagnostics and lab workflows.  
Developed using Java Swing, MySQL, and JasperReports.

## Features
- Secure login & authentication
- Patient management with demographics
- Test catalog with pricing & specifications
- Automated result processing & PDF reports
- Audit trails & error handling

## Tech Stack
Java | Swing | MySQL | JasperReports | JDBC | NetBeans IDE

## Impact
✔ Reduced manual errors by 25%  
✔ Improved report generation speed by 30%  
✔ Centralized patient records for better care
